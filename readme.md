# The Default Planck Layout

### Instructions

Compile the keymap:

`make planck/rev6:jm`

Upload output file to qmk toolbox (`planck_rev6_jm.bin` in root dir)

Start keyboard in bootloader mode:

- Unplug keyboard
- Hold space and B
- Plug keyboard in and wait a bit
- Press flash
- Unplug keyboard and plug back in
